
##Pytorch Implementation of dmlab30 language model

### Usage

```shell
$ git clone https://github.com/Shengyu-Feng/deepmind_lab_language
$ cd lab
$ bazel run :learning_agent
```
Agent code is in the folder [python/rl_agent](python/rl_agent)
